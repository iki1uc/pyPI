# pyPI — MASTER

pyPI ist das zentrale Interpreter-Modul des Systems.
Es setzt **ID**, **INDEX**, bindet **NC36**, aktiviert die **tp‑Achse**
und schaltet **ULRM** frei. Erst danach kann **LAGE** wieder MASTER werden.

---
## ID / INDEX
- `ID.html`   = Identität des Moduls  
- `index.html` = Einstiegspunkt + tp‑Achse + Motorlinie  

Beide Dateien müssen vorhanden sein, bevor pyPI MASTER wird.
